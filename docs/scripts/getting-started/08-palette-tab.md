# Palette Tab

Goal

- Store and recall voicings.

What you’ll do

- Trigger cymatic buttons, manage expressions, and understand banks.

Steps

1. Open Palette Tab → click a cymatic to trigger a voicing.
2. Open the expression window; duplicate an expression; switch expressions.
3. Note bank organization for larger palettes.

Tip

- Palettes store voicings (cymatics/expressions); they are reusable across songs.
