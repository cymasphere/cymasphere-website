# Topbar Navigation

Goal

- Find global controls fast and switch views quickly.

What you’ll do

- Identify the Topbar and use it to switch between core tabs; access Help via the User Menu.

Steps

1. Locate the Topbar at the top of the window.
2. Click through tabs: Song → Track → Palette → Voicing.
3. Open the User Menu on the right and note Help/About.

Tip

- Use the Topbar for navigation; do edits inside each tab.
