# Layer Manager

Goal

- Organize and protect your composition layers.

What you’ll do

- Toggle visibility, reorder layers, and lock layers to avoid accidental edits.

Steps

1. Open the Layer Manager.
2. Toggle layer visibility to focus on specific parts.
3. Drag to reorder layers as needed.
4. Lock a layer you don’t want to change.

Tip

- If you can’t edit a part, check whether the layer is locked.
